Overview

Purpose

The purpose of this website is to describe my joy of the toki pona language. The main website contains a description and a few images of the language. One page contains several text-boxes which render sitelen pona (the wrinting system) in different fonts. Another page is a comprehensive resource and description of multiple learning sources.

Audience

Everyone who likes constructed languages, or are interested in learning toki pona.
